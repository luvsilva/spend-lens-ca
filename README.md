# SpendLens Canada 💸

An AI-powered bank statement analyzer designed for the Canadian market.

## ✨ Features
- **AI Analysis:** Uses Gemini 2.5 Flash to categorize transactions.
- **Privacy First:** Client-side processing. Your financial data is never stored.
- **Support:** Optimized for CIBC, TD, RBC, and Scotiabank CSV formats.

## 🚀 How to use
1. Get your API Key from [Google AI Studio](https://aistudio.google.com/).
2. Upload your Canadian bank CSV statements.
3. Get instant insights and spending charts.

## 🛠️ Tech Stack
- HTML5 / Tailwind CSS
- Chart.js
- Google Gemini API
